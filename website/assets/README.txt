Place the following files in this directory before deploying:

  favicon.png   — 32×32 app icon (teal crescent on white or transparent)
  logo.png      — Full Noor Companion wordmark (for email templates etc.)
  og-image.png  — 1200×630 Open Graph social sharing image
                  Suggested: teal background, white crescent + wordmark, tagline

These files are gitignored to keep binary assets out of the repo.
Use Supabase Storage or a CDN for production assets if preferred.
